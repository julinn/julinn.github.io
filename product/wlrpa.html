核心功能

随机延迟 · 模拟真人节奏

支持设置最小/最大延迟时间范围，每次操作前在范围内随机等待，避免固定间隔被识别为机器行为，更贴近真人操作习惯，操作节奏自然不僵硬。

随机位置点击 · 防封更自然

可指定屏幕目标区域，在范围内随机选取坐标进行点击，配合随机延迟让点击轨迹更自然，有效规避平台风控策略，降低账号异常风险。

CSV数据驱动 · for循环批处理

支持导入CSV表格，逐行读取数据（如账号、昵称、话术、商品信息），通过for循环批量执行任务，一次配置即可自动完成成百上千条重复操作。

模拟按键 · 覆盖全键盘操作

完整模拟键盘输入与组合键（如 Ctrl+C / Ctrl+V、Enter、方向键、Tab 等），可编排多步按键序列，适用于表单填写、消息发送、快捷操作等场景。

百度文字识别 · 屏幕文字自动读取

集成百度OCR能力，可指定截取屏幕区域并识别其中文字，将识别结果用于条件判断或数据回填，实现真正"看得见"的智能自动化。

图形化配置 · 附带指令说明和样例，轻松上手

参数与流程通过可视化界面配置，无需专业代码能力；普通运营人员也能快速搭建属于自己的自动化流程。

百度文字识别（OCR）配置说明

  • 注册并登录 百度智能云（cloud.baidu.com）账号，完成实名认证；
  • 进入「文字识别」控制台，开通「通用文字识别（标准版 / 高精度版）」服务；
  • 在「应用列表」中创建应用，获取 API Key 与 Secret Key；
  • 在软件「参数设置 - 百度OCR」中填入上述 Key 并保存即可启用；
  • 免费额度用尽后将按百度官方计费标准收费，请关注用量避免服务中断。

    • 注意：使用百度文字识别功能前，需提前开通百度OCR服务并配置相关参数，否则该功能将无法正常工作。
